### 1-6. 모션 (Motion / Transition)

| 토큰                 | 값                                  | 사용처                  | 규칙                   |
| -------------------- | ----------------------------------- | ----------------------- | ---------------------- |
| `--duration-instant` | 100ms                               | 체크박스 토글           |                        |
| `--duration-fast`    | 200ms                               | 버튼 hover/active       | 기본 마이크로 인터랙션 |
| `--duration-normal`  | 300ms                               | 카드 전환, 탭 전환      |                        |
| `--duration-slow`    | 500ms                               | 모달 진입/퇴장          |                        |
| `--duration-slide`   | 600ms                               | 대기 화면 슬라이드 전환 |                        |
| `--easing-default`   | `cubic-bezier(0.4, 0, 0.2, 1)`      | 대부분의 전환           | Material ease          |
| `--easing-enter`     | `cubic-bezier(0, 0, 0.2, 1)`        | 요소 진입               | 감속 커브              |
| `--easing-exit`      | `cubic-bezier(0.4, 0, 1, 1)`        | 요소 퇴장               | 가속 커브              |
| `--easing-spring`    | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 바운스 피드백           | 터치 피드백 전용       |

**규칙:**

- `prefers-reduced-motion: reduce` 미디어 쿼리 감지 시 모든 duration을 0ms로 재정의.
- `transform` 및 `opacity` 속성만 애니메이션. `width`, `height`, `top`, `left` 애니메이션 금지.
- 대기 화면 슬라이드: 5초 간격 자동 전환, 터치 시 즉시 메인 화면 이동.

## 2. 레이아웃 (Layout)

### 2-1. 해상도 및 브레이크포인트

> 키오스크 전용 시스템이므로 반응형 브레이크포인트 대신 **고정 기준 해상도**를 사용한다.

| 항목         | 값                     | 규칙                                |
| ------------ | ---------------------- | ----------------------------------- |
| 기준 해상도  | 1920 × 1080 px (FHD)   | 모든 레이아웃은 이 해상도 기준 설계 |
| 보조 해상도  | 3840 × 2160 px (4K)    | 43인치 패널 대비, 2× 스케일링 적용  |
| 최소 해상도  | 1366 × 768 px          | 저가형 패널 폴백                    |
| 방향         | 가로(Landscape) 고정   | 세로 모드 사용 금지                 |
| DPI 스케일링 | 기준 1× (FHD), 2× (4K) | CSS `rem` 기반 스케일링             |

### 2-2. 그리드 & 컨테이너

| 항목                 | 값           | 규칙                            |
| -------------------- | ------------ | ------------------------------- |
| 컨테이너 최대 너비   | 1760px       | 좌우 80px 여백 (1920−160)       |
| 컨테이너 사이드 패딩 | 48px 고정    | `--space-48` 적용               |
| 메인 그리드          | 12 컬럼      | 거터(gutter) 32px               |
| 메인 메뉴 그리드     | 4 × 2 (8칸)  | 카테고리 8개를 4열 2행으로 배치 |
| 메뉴 카드 크기       | 380 × 400 px | 터치 영역 충분 확보             |
| 메뉴 카드 간격       | 32px         | `--space-32`                    |

### 2-3. 영역 분할

```
┌─────────────────────────────────────────────────────────┐
│  헤더 (고정)  │  높이: 80px  │  로고 + 현재 시간 + 관리자 버튼  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    콘텐츠 영역                            │
│              (스크롤 가능, 높이: calc(100vh - 80px))       │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  하단 바 (고정)  │  높이: 64px  │  홈·뒤로·확대/축소 컨트롤  │
└─────────────────────────────────────────────────────────┘
```

| 영역    | 높이                  | 위치        | 규칙                                                  |
| ------- | --------------------- | ----------- | ----------------------------------------------------- |
| 헤더    | 80px                  | 상단 고정   | `position: fixed; top: 0`                             |
| 콘텐츠  | `calc(100vh - 144px)` | 스크롤 영역 | `overflow-y: auto; -webkit-overflow-scrolling: touch` |
| 하단 바 | 64px                  | 하단 고정   | `position: fixed; bottom: 0`                          |

### 2-4. 스크롤 & 고정 규칙

| 항목             | 규칙                                                      |
| ---------------- | --------------------------------------------------------- |
| 콘텐츠 스크롤    | 터치 드래그로 자연스러운 관성 스크롤                      |
| 스크롤바         | 기본 숨김, 스크롤 중에만 6px 둥근 인디케이터 표시         |

| 멀티터치 줌      | 지도·도면 뷰어에서만 허용 (pinch-to-zoom)                 |

### 3-1. 대기 화면 (스크린세이버)

| 항목          | 값                                             | 규칙                                           |
| ------------- | ---------------------------------------------- | ---------------------------------------------- |
| 배경          | `--screensaver-bg` (#0A1628)                   | 풀스크린, 상태바 숨김                          |
| 슬라이드 전환 | 5초 간격, crossfade 600ms                      | `--duration-slide` 적용                        |
| 콘텐츠        | 안전 포스터 이미지(JPG/PNG) 또는 슬로건 텍스트 | `/content/screensaver/` 디렉터리에서 자동 로드 |
| 하단 안내     | "화면을 터치하세요" 텍스트 (펄스 애니메이션)   | `--typo-h2`, `--screensaver-text`              |
| 진입 조건     | 120초 미조작 시 자동 전환                      | 관리자 모드에서 시간(초) 변경 가능             |
| 복귀 조건     | 아무 곳 터치 시 메인 화면으로 전환             | fade-out 300ms → 메인 화면 fade-in 300ms       |

### 3-2. 메인 화면 (카테고리 그리드)

| 항목      | 값                                                              | 규칙                                |
| --------- | --------------------------------------------------------------- | ----------------------------------- |
| 레이아웃  | 4열 × 2행 그리드                                                | 총 8개 카테고리                     |
| 카드 크기 | 380 × 400 px                                                    | 최소 터치 타깃 44px 초과 달성       |
| 카드 구성 | 아이콘(96×96) + 카테고리명(h3) + 부가 설명(caption)             | 수직 중앙 정렬                      |
| 아이콘    | SVG, 단색 `--color-primary`                                     | Lucide 또는 커스텀 SVG              |
| 상단 영역 | 로고(좌) + "당진발전본부 안전관리" 타이틀(중앙) + 현재 시각(우) | 헤더 내 배치                        |
| 하단 바   | "홈" 버튼 (좌) + "관리자" 잠금 아이콘 (우)                      | 관리자 버튼은 5초 롱프레스로 활성화 |

### 3-3. 콘텐츠 뷰어 화면

| 콘텐츠 타입      | 뷰어                | 규칙                                          |
| ---------------- | ------------------- | --------------------------------------------- |
| PDF              | PDF.js 기반 렌더링  | 페이지 단위 스크롤, 상·하 페이지 이동 버튼    |
| 이미지 (JPG/PNG) | `<img>` + 핀치 줌   | 멀티터치 줌인/줌아웃, 더블탭 → 원본 비율 복귀 |
| 동영상 (MP4)     | HTML5 `<video>`     | 자동 재생 금지, 재생/일시정지/전체화면 컨트롤 |
| 지도/도면        | 캔버스 기반 패닝+줌 | 최소 0.5× ~ 최대 3× 줌, 더블탭 → 1× 복귀      |

**공통 규칙:**

- 뒤로 가기: 좌측 상단 화살표 + 하단 바 "뒤로" 버튼 동시 제공.
- 타이틀: 현재 카테고리명 ▸ 문서 제목 (breadcrumb).
- 60초 미조작 시 "계속 보시겠습니까?" 확인 모달 표시. 10초 후 자동 메인 복귀.

| 스타일      | 배경              | 텍스트            | 테두리                      | 사용처                 |
| ----------- | ----------------- | ----------------- | --------------------------- | ---------------------- |
