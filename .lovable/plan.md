

# 당진발전본부 안전관리 키오스크 — 구현 계획

## 1단계: 디자인 시스템 구축
상세서에 정의된 디자인 토큰(색상, 타이포그래피, 간격, 라운딩, 그림자, 모션, z-index)을 CSS 변수와 Tailwind 설정에 적용합니다. Pretendard 폰트를 로컬 번들로 설정합니다.

## 2단계: 레이아웃 & 공통 컴포넌트
- **AppLayout**: 고정 헤더(80px, 로고+타이틀+시각표시) + 콘텐츠 영역 + 고정 하단바(64px, 홈/뒤로/관리자)
- **KioskButton**: lg/md/sm 크기, primary/secondary/danger/ghost/icon-only 스타일, 상태(hover/active/disabled/loading)
- **KioskCard**: 380×400 카테고리 메뉴 카드 (아이콘 96px + 제목 + 부제)
- **KioskModal**: 480~640px, 오버레이, 포커스 트랩, 진입/퇴장 애니메이션
- **KioskDrawer**: 우측 480px 슬라이드 드로어
- **KioskToast**: 하단 중앙, 4타입(success/warning/error/info), 자동 소멸 4초
- **KioskTable**: 관리자 전용 테이블, zebra 패턴, 정렬, 페이지네이션

## 3단계: 화면 구현

### 대기 화면 (스크린세이버)
- 다크 배경(#0A1628), 이미지 슬라이드 5초 간격 crossfade
- "화면을 터치하세요" 펄스 애니메이션
- 120초 미조작 시 자동 전환, 터치 시 메인 화면으로 복귀

### 메인 화면
- 4열×2행 그리드에 8개 카테고리 카드 배치
  - 본부 지도, 안전관리 수칙, 보건관리 정보, 사고 사례 전파, 비상시 대응 조치, 긴급 위험 신고 안내, + 2개 추가
- 헤더: 로고 + "당진발전본부 안전관리" + 현재 시각
- 하단바: 홈 버튼 + 관리자 아이콘(5초 롱프레스로 진입)

### 콘텐츠 뷰어
- 카테고리 선택 시 해당 콘텐츠 목록 표시
- PDF, 이미지, 동영상 뷰어 지원
- 브레드크럼 네비게이션 (카테고리명 ▸ 문서 제목)
- 60초 미조작 시 "계속 보시겠습니까?" 확인 모달 → 10초 후 자동 메인 복귀

### 관리자 모드
- 6자리 PIN 입력 (가상 넘버패드), 5회 실패 시 300초 잠금
- 좌측 사이드바(240px): 카테고리 1~8, 시스템 설정
- 콘텐츠 관리: 테이블 뷰, 파일 업로드/삭제, 미리보기
- 300초 미조작 시 자동 로그아웃

## 4단계: Supabase 백엔드
- **categories 테이블**: 8개 카테고리 (이름, 아이콘, 정렬순서, 설명)
- **contents 테이블**: 카테고리별 콘텐츠 (제목, 파일 URL, 타입, 카테고리 FK)
- **screensaver_slides 테이블**: 대기 화면 슬라이드 이미지
- **settings 테이블**: 시스템 설정 (타임아웃, PIN 등)
- **Storage 버킷**: 콘텐츠 파일 저장 (PDF, 이미지, 동영상)
- 관리자 인증을 위한 PIN 검증 Edge Function

## 5단계: 상호작용 & 접근성
- 터치 리플 효과 (반경 40px, 300ms)
- 화면 전환 애니메이션 (crossfade, slide)
- prefers-reduced-motion 대응
- 시맨틱 HTML, aria-label, 포커스 관리
- 자동 스크롤 복귀 (60초 미조작)
